Welcome to Pyautoupdate’s documentation!¶

Pyautoupdate is an auto-update API for Python programs. Developers can focus on developing their application and use Pyautoupdate to easily ensure that the copy of the code on the end user’s computer remains updated.

Example code using Pyautoupdate can be found in Quickstart.

Pyautoupdate is released under the LGPL 2.1.
